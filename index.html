Mastering Terraform State

July 30, 2025

Today I dove into Chapter 3 of 'Terraform: Up & Running', focusing on state management. The concepts of remote state, locking, and isolation are critical for team collaboration. Storing state in S3 and using DynamoDB for locking prevents conflicts and is a huge step up from local state files. This was a key lesson in moving from solo projects to professional-grade infrastructure.

Building a Container from Scratch

July 28, 2025

Worked through the `rubber-docker` project to understand what a container really is. It's not magic! It's a combination of Linux namespaces (for isolation) and cgroups (for resource limits). Using `pivot_root` instead of `chroot` to create a secure filesystem jail was a major 'aha!' moment. This project was challenging but incredibly rewarding.
